# Power BI Dashboards

## Overview
Interactive dashboards enabling dimensional analysis of student performance across demographics, geography, assessment types, and time periods.

## Files

**[dashboard-overview.pdf](dashboard-overview.pdf)**
Complete dashboard export including performance visualizations, key insights, and strategic recommendations.

**[data-model-diagram.png](data-model-diagram.png)**
Power BI data model showing dimensional structure and table relationships.

## Key Features

- Performance trends over time across all assessment types
- Demographic breakdowns by gender and year level
- Geographic analysis (urban vs remote regions)
- Interactive drill-down and cross-filtering capabilities
- Executive summary of findings and recommended actions

## Technologies

Power BI | DAX | Power Query
